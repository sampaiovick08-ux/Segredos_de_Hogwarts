🎯 1. Visão do Produto

O que é o seu programa?
É um jogo inspirado nas crônicas de Harry Potter, com o objetivo de vivenciar um pouco da experiência dos livros/filmes.



📦 2. O MVP (Minimum Viable Product)

Quais são as 5 funcionalidades PRINCIPAIS e OBRIGATÓRIAS do sistema?

O usuário irá participar de um quiz para selecionar sua casa;

O usuário poderá escolher de qual aula irá participar;

O usuário irá participar de uma batalha.

O usuário irá ter que resolver uma charada;

O usuário participará da batalha final.

🚫 3. Fora de Escopo

O que o sistema NÃO vai fazer (para não perdermos tempo)?

O caminho que terá que percorrer;

Não mostrará como o personagem chegou até a escola.

🗺️ 5. Fluxo do Usuário (Passo a Passo)

O usuário abre o jogo

O usuário inicia o jogo

O sistema pergunta "qual seu nome?"

O usuário responde 

O sistema pergunta "qual seu sobrenome?"

O usuário responde 

Irá ocorrer a seleção das casas

O personagem terá que escolher entre 2 professores para ter aula 

O personagem terá aula com o professor escolhido 

Lá ele conhecerá um rival

Após a aula ele encontrará um monstro 

Ele batalha com esse monstro 

Após matar o monstro, ele vai tentar descobrir quem o enviou 

Ele resolve charadas e é mandado até o porão 

Ele descobre quem mandou o monstro e batalha com a pessoa

E descobre o segredo
